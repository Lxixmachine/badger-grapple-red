# Roadmap

## Milestone 1 — Inventory
- Scan all map directories.
- Store provenance and dimensions.
- Validate normalized records.

## Milestone 2 — Geometry
- Decode map block data.
- Resolve metatile IDs.
- Build rendered base maps.
- Segment into camera screens.

## Milestone 3 — Semantics
- Decode metatile collision and behavior.
- Identify traversable components.
- Construct navigation graphs.
- Import events, warps, objects, and connections.

## Milestone 4 — Analysis
- Compute path-width distributions.
- Compute open-space and obstacle ratios.
- Detect decision nodes and chokepoints.
- Add landmark visibility annotations.
- Run composition and grid-exposure review.

## Milestone 5 — Encyclopedia
- Generate measured case studies.
- Compare maps by functional role.
- Update Bible rules when evidence contradicts them.

## Non-goals
- Redistributing ROMs.
- Redistributing proprietary game graphics.
- Claiming unsupported measurements.
- Training a model directly on copyrighted assets without legal review.

# 1. Pattern Taxonomy

Patterns are grouped into:

- Entry
- Connector
- Landmark
- Destination
- Reward
- Boundary
- Transition
- Compression
- Release
- Secret

Each pattern solves a recurring spatial problem.

---

# 2. Town Patterns

## PAT-101 Main Street

Purpose:
Create a readable spine.

Rules:
- Connect major buildings.
- Maintain sightline to one landmark.
- Branch sparingly.

---

## PAT-102 Residential Cluster

Purpose:
Create neighborhood identity.

Characteristics:
- Shared spacing language
- Small variations
- One local landmark

---

## PAT-103 Civic Center

Purpose:
Anchor town hierarchy.

Contains:
- Gym, town hall, plaza, monument, or equivalent.

---

## PAT-104 Market Edge

Purpose:
Transition from civic core to quieter residential space.

---

# 3. Route Patterns

## PAT-201 Scenic Bend

Long curve revealing a distant landmark.

## PAT-202 Trainer Pocket

Small widened space reserved for an encounter.

## PAT-203 Safe Clearing

Low-density area after challenge.

## PAT-204 Optional Loop

Reconnects to main route after optional exploration.

## PAT-205 Vista Reveal

Destination becomes visible before it is reachable.

---

# 4. Terrain Patterns

## PAT-301 Forest Wall

Continuous tree mass with varied edge depth.

## PAT-302 River Corridor

Water guides movement while defining boundary.

## PAT-303 Cliff Shelf

Raised resting area overlooking lower terrain.

## PAT-304 Meadow Basin

Open low-density field surrounded by stronger edges.

# 5. Landmark Patterns

## PAT-401 Lone Tree

Single dominant natural landmark.

## PAT-402 Monument Plaza

Landmark defines public gathering space.

## PAT-403 Watchtower

Visible from multiple screens.

## PAT-404 Ancient Ruin

Communicates history and exploration.

# 6. Secret Patterns

## PAT-501 Hidden Item Pocket

Visible only after leaving main flow.

## PAT-502 False Dead End

Appears empty but rewards investigation.

## PAT-503 Hidden Shortcut

Later unlocks faster traversal.

Rules:
- Never block progression.
- Always reward curiosity.

---

# 7. Transition Patterns

## PAT-601 Town Gate

Signals movement between identities.

## PAT-602 Bridge Threshold

Compression before crossing.

## PAT-603 Forest Mouth

Gradual shift from civilization to wilderness.

## PAT-604 Cave Entrance

Reduce visibility while increasing anticipation.

---

# 8. Pattern Composition

Recommended chains:

Town
→ Gate
→ Connector
→ Scenic Bend
→ Trainer Pocket
→ Safe Clearing
→ Landmark
→ Destination

Avoid repeating identical chains without variation.

---

# 9. Pattern Selection Matrix

| Goal | Recommended Patterns |
|------|----------------------|
| Teach exploration | Optional Loop, Hidden Item Pocket |
| Increase tension | Forest Wall, Cave Entrance |
| Reward player | Vista Reveal, Cliff Shelf |
| Create identity | Monument Plaza, Lone Tree |
| Slow pacing | Scenic Bend, Meadow Basin |

# 10. AI Pattern Assembly

Workflow:

1. Determine gameplay purpose.
2. Choose compatible patterns.
3. Order by rhythm.
4. Validate against previous volumes.
5. Generate blockout.
6. Audit.
7. Decorate.

Required Output:

```markdown
Pattern Chain
Purpose
Satisfied Rules
Potential Risks
Suggested Revisions
```

# Final Principle

Players rarely remember individual tiles.

They remember spatial experiences.

Patterns are the reusable building blocks of those experiences.

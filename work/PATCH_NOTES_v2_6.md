# Badger Grapple Red Engine v2.6

## Improved
- Added an overworld objective tracker so the player knows what to do next.
- Added save normalization so older phone saves do not break newer builds.
- Added demo stats: scouts, battles, wins, recruits joined.
- Removed random auto-scout interruption; scout zones now trigger by pressing A.
- Scout reports now show moves and a style-tip before the challenge.
- Battle command panel now shows selected move power, accuracy, gas cost, point value, and style matchup.
- Battle resets match score at fight start.
- Recruit outcomes now set a clear return message on the overworld.
- Deployed build imports `./src_v2_6/main.js?v=26` to avoid mobile cache confusion.

# v17.9 FireRed Battle Restore

- Restored the Pokemon FireRed-style battle command flow.
- Battle now opens on the classic command menu: FIGHT / MOVE / WRESTLER / RUN.
- FIGHT opens the wrestler move list instead of immediately showing moves as the main screen.
- WRESTLER opens the party switch panel.
- RUN exits wild/scout battles but not captain battles.
- Preserved Phaser, mobile controls, title, save/load, overworld, Field House sparring, scouting, recruiting, and asset-path repair.
- No canvas fallback or engine replacement.

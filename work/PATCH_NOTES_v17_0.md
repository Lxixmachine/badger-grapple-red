# v17.0 — Mockup Promise Pass

This build moves the project from a single Field House shell toward the mockups:

- Seven connected areas: Field House, Campus Quad, Lakeshore, Downtown, River Trail, Conference Arena, Championship Hall.
- New full-map pixel backdrops for every area instead of one repeated scene.
- Area-to-area transitions with persistent area/position saving.
- Scout grass in Campus/Lakeshore/River areas, with area-biased wrestler pools.
- Recovery/shop/NPC/captain interaction markers.
- More commercial-GBA style top HUD, bottom prompt, fade transitions, and denser backgrounds.

Still not final FireRed-level production art. The next bottleneck is hand-authored sprite sheets, animated tiles, and sound.

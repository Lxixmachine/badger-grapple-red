# Bucky's Locker Room Service Kit V1

Built with ChatGPT Image Generation as an original equipment-shop
production board. The FireRed Center/Mart sheet was used only to calibrate
handheld pixel density, value grouping, and readable silhouettes.

The six requested modules were: a navy/cream/cardinal north wall, matching
side wall, south threshold wall, seven-cell sales counter, three-cell singlet
display, and three-cell athletic-supply display. All modules use a flat
`#ff00ff` chroma-key field, one-pixel tinted outlines, simple material ramps,
no text or logos, and no copied game assets.

The deterministic source regions and logical sizes are declared in
`tools/prepare_imagegen_tileset_sources.py`.

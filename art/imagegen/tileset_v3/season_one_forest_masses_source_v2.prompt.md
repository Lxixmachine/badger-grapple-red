# Season One forest masses source v2

Built with the built-in ChatGPT ImageGen tool as a controlled edit of the v1
4x3 production board. The panel order, dimensions, silhouettes, joining edges,
and flat magenta key remain unchanged.

This revision shifts the foliage to a brighter medium-green three-shade ramp,
reserves deep blue-green for thin outlines and small recesses, removes
black-looking canopy voids, and reduces painterly micro-noise. It keeps the
same grid-native forest assemblies while improving value separation and
phone-scale readability against the FireRed study reference.

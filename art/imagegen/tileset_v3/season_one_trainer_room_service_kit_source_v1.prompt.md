# Trainer's Room Service Kit V1

Built with ChatGPT Image Generation as an original 3x2 service-interior
production board. The FireRed Center/Mart sheet was used only to calibrate
handheld pixel density, value grouping, and readable silhouettes.

The six requested modules were: a cream/teal/cardinal north wall, matching
side wall, south threshold wall, seven-cell recovery counter, three-cell
roster terminal, and three-cell athletic treatment table. All modules use a
flat `#ff00ff` chroma-key field, one-pixel tinted outlines, simple material
ramps, no text or logos, and no copied game assets.

The deterministic source regions and logical sizes are declared in
`tools/prepare_imagegen_tileset_sources.py`.

# Season One Forest Clusters Source v1

- Tool: built-in ChatGPT Imagegen
- Mode: new raster asset generation
- Runtime intent: six original broad forest modules normalized to `32x48` logical pixels (`2x3` map cells)
- Ownership: the deterministic compositor controls placement, overlap, collision, and whole-cell clipping

## Prompt

Use case: stylized-concept

Asset type: source board for an original grid-native 2D RPG forest metatile system

Primary request: Create a clean 3-column by 2-row board containing exactly six separate broad deciduous shade-tree modules. Each module is designed to occupy exactly two 16-pixel logical cells wide by three 16-pixel logical cells tall after downsampling (2:3 module aspect). Row 1 contains three dense canopy-led back/interior forest modules with the trunk mostly hidden. Row 2 contains three front-edge modules with a compact visible central trunk and roots at the bottom. Exactly one tree module per panel.

Scene/backdrop: perfectly flat, uniform solid #ff00ff magenta across the entire board, including gutters; no shadows or texture in the background.

Style/medium: crisp original Game Boy Advance-era orthographic pixel art, deliberately placed hard-edged pixel clusters, broad layered rounded crowns with slightly irregular silhouettes, dark teal-green tinted outline, exactly three compatible foliage shades, compact warm-brown trunks.

Composition/framing: equal 3x2 panel spacing with generous clean gutters; every crown nearly fills its panel width and height while staying fully separated from neighboring panels; consistent scale and lighting across all six.

Constraints: broad two-cell tree scale; rounded deciduous shade trees; compatible interlocking shoulders; one subject per panel; no copied Pokemon pixels, layouts, or recognizable proprietary designs; no text; no panel borders; no grid lines; no cast shadows; no watermark.

Avoid: narrow one-cell trees, topiary balls, conifers, sharp triangular peaks, giant multi-tree blobs, painterly noise, blur, gradients, antialiasing, transparent background, grass or ground planes.

# Season One forest masses source v1

Built with the built-in ChatGPT ImageGen tool as a dedicated 4x3 production
board. Each panel is one complete grid-native forest assembly rather than a
single tree repeated by the map compositor.

Panel order: north edge A/B, south edge A/B, west edge A/B, east edge A/B,
6x8 core, 5x4 grove, 5x5 inner northwest corner, and 6x5 outer southwest
corner. All pieces share one GBA-era foliage ramp, hard pixel clusters, a flat
magenta key, and explicit seamless joining edges. The board forbids isolated
tree rows, wallpaper bands, exposed-trunk stripes through forest bodies, text,
gradients, antialiasing, and watermarks.

# Trainer Floor Inlay V1

Built with ChatGPT Image Generation as one original, walkable three-by-three
cell floor medallion. It uses a cream outer wrestling circle, thin cardinal
boundary rings, and a small teal center mark on a flat `#ff00ff` field.

The prompt required transparent negative space, no height or shadow, no text,
logos, characters, branded imagery, or copied game assets. The runtime collision
mask is fully walkable.

# Lakeshore Path Full Composition v1

Use case: stylized-concept
Asset type: source painting for a 56x14 tile top-down RPG route
Primary request: Create an original polished GBA-era pixel-art map composition for Madison's Lakeshore Path. Use FireRed only as a readability and map-design rubric; do not copy any Nintendo pixels, buildings, or layouts. The final map is a long 4:1 route band with Lake Mendota across the entire north edge, a quiet pale shore path across the middle, dense forest along the south edge, three unmistakable worn cardinal-red wrestling mats, and one memorable center marina cluster: a pier extending north into the lake, three Terrace-style chairs, and a locked boathouse. Add a few benches, trees, flowers, rocks, and lamps as secondary anchors while preserving broad quiet walking lanes. Every barrier and doorway must align naturally to a 16x16 gameplay grid after resizing. No characters, labels, UI, logos, gradients, blur, or atmospheric perspective. Pixel art must remain crisp, saturated, readable, and original.

Output framing: exact 4:1 visible composition surrounded by a flat removable magenta border; no cropped route edges.
